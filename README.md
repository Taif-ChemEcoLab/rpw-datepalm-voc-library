# RPW–Date Palm VOC Library

This repository contains a structured VOC dataset generated from GC-MS analysis of date palm samples under varying Red Palm Weevil (RPW) infestation conditions.

## Objective

The dataset has been standardized and transformed into a machine-learning-ready format to support:

- Early detection biomarker discovery  
- Infestation state classification  
- Comparative VOC fingerprint analysis  
- Feature ranking and compound importance modeling  
- Integration with biosensor or field detection systems  

## Reproducibility

All data cleaning, compound harmonization, and matrix generation steps are fully reproducible through the provided scripts.
